# 🚀 Personal Portfolio Website (Full Stack)

A full-stack personal portfolio website built using **HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB**.  
This project showcases my skills, projects, and contact information with dynamic backend integration.

---

## 🌐 Live Demo

- 🔗 Frontend: https://personal-portfil.netlify.app/  
- 🔗 Backend API: https://your-backend-url.onrender.com

---

## 📌 Features

- Responsive modern UI design
- About section with personal introduction
- Skills showcase section
- Dynamic Projects section (from backend API)
- Contact section with email & social links
- Full-stack integration (Frontend + Backend + Database)

## ⚙️ How It Works

1. User opens portfolio website
2. Frontend loads HTML/CSS/JS
3. JavaScript calls backend API `/projects`
4. Backend fetches data from MongoDB
5. Data is displayed dynamically on UI
